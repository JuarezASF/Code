-para compilar da pasta atual e colocar o .class em bin o
comando é:

    javac ./src/AloMundo.java -d ./bin



-para executar o .class a partir do diretorio atual o comando
é:

    java -cp bin/ AloMundo

*acredito que -cp seja para class path

-comando para ver as mensagens do compilador:
    -verbose
*mesmo quando da certo ele lista diversas mensagens
